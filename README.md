## Getting started
 1) Clone and install `strategoutil` to your environment

```sh
git clone https://github.com/mihsamusev/strategoutil.git
cd strategoutil
pip install .
```

2) Run tests
```sh
python -m pytest
```

3) Import to your stratego MPC project and start hacking, see [example project]()
```python
import strategoutil
```
